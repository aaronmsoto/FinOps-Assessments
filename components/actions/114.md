# 114: Collect Standardized Assessment Inputs

**Slug:** 114-collect-standard

**Implementation Types:**

## Descriptive Statement

Collect evidence-based inputs; reduce bias with standardized questions.

## Supplemental Guidance

As an example of consistent assumptions, let's look at action 010.
Here, an assumption could be what is considered a shared cost.
One scoped team may include networking and data transfer, while another may include security infrastructure.
Thus, these assumptions should be defined for all teams before evaluating separately scoped assessments.

## Scoring Guide

**Weight:** 1.0

### Formula

* Use objective (not subjective) scoring metrics for assessment evaluation
* Document how metrics used in the assessment are calculated and state all assumptions
* Use the same calculation assumptions and formulas across scopes if possible
* List relevant documentation of process and policy as evidentiary support for each score
